### 方法二：本地服务器（推荐） ⭐⭐⭐

#### 使用Python（最简单）

```bash
# 进入项目目录
cd /path/to/project

# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后在浏览器打开：`http://localhost:8000/home.html`

#### 使用Node.js

```bash
# 安装http-server（首次使用）
npm install -g http-server

# 启动服务器
http-server -p 8000
```

然后在浏览器打开：`http://localhost:8000/home.html`

#### 使用PHP

```bash
php -S localhost:8000
```

然后在浏览器打开：`http://localhost:8000/home.html`

**适用场景：**
- 开发和测试
- 完整功能体验
- 避免跨域问题

## PDF OCR 工具使用

### 第一步：打开工具

选择以下任一方式：
- 打开 `pdf-ocr.html`
- 从 `home.html` 点击"PDF OCR 工具"卡片

### 第二步：上传PDF文件

**方式A：拖拽上传**
1. 准备一个PDF文件
2. 直接拖拽到上传区域
3. 松开鼠标即可开始处理

**方式B：点击上传**
1. 点击"选择PDF文件"按钮
2. 在文件选择对话框中选择PDF
3. 点击"打开"开始处理

### 第三步：等待处理

系统会自动：
- 📖 读取PDF内容
- 🖼️ 将每页转换为图片
- 🔍 识别图片中的文字
- 📊 显示实时进度

处理时间：
- 单页简单文档：5-10秒
- 单页复杂文档：10-20秒
- 多页文档：每页约10-20秒

### 第四步：查看结果

处理完成后会显示：
- ✅ 识别的文字内容
- 📄 总页数
- 🔢 识别字数
- ⏱️ 处理时间

### 第五步：导出结果

**选项A：复制文本**
1. 点击"📋 复制文本"按钮
2. 在任意地方粘贴（Ctrl+V 或 Cmd+V）

**选项B：下载TXT文件**
1. 点击"💾 导出为TXT"按钮
2. 文件会自动下载到默认下载文件夹
3. 文件名格式：`PDF识别结果_2024-12-25T12-30-45.txt`

### 第六步：继续使用

- 点击"🔄 重新上传"处理新的PDF
- 或关闭页面结束使用

## 生日贺卡使用

## 常见问题

### PDF OCR 相关

#### Q1: 为什么首次使用很慢？
**A:** 首次使用需要下载OCR引擎和语言包（约10-20MB），之后会缓存到浏览器，速度会快很多。

#### Q2: 识别结果不准确怎么办？
**A:** 识别准确度取决于PDF质量，建议：
- 使用清晰的扫描文档
- 确保文字不倾斜
- 避免背景噪点
- 使用高对比度文档

#### Q3: 支持哪些语言？
**A:** 当前支持中文（简体）和英文。如需其他语言，可以修改代码中的语言包配置。

#### Q4: PDF文件会上传到服务器吗？
**A:** 不会！所有处理都在你的浏览器本地完成，文件不会离开你的电脑，完全保护隐私。

#### Q5: 可以处理加密的PDF吗？
**A:** 不支持。请先使用其他工具移除PDF密码保护。

#### Q6: 手机上可以使用吗？
**A:** 可以！但建议在WiFi环境下使用，处理大文件可能比较慢。

### 生日贺卡相关

#### Q1: 音乐为什么不自动播放？
**A:** 现代浏览器默认阻止自动播放音频。用户需要点击页面后才能播放。这是浏览器的安全策略。

#### Q2: 可以修改祝福文字吗？
**A:** 可以！打开 `index.html` 文件，找到 `<h1 class="birthday-text">生日快乐！</h1>` 这行，修改文字内容即可。

#### Q3: 怎么改变颜色主题？
**A:** 打开 `styles.css` 文件，修改渐变色配置。详见README.md的自定义建议部分。

### 通用问题

#### Q1: 需要安装什么软件吗？
**A:** 不需要！只需要现代浏览器即可。

#### Q2: 可以离线使用吗？
**A:** 
- 生日贺卡：完全离线可用
- PDF OCR：首次需要联网下载语言包，之后可离线使用

#### Q3: 数据会被保存吗？
**A:** 不会自动保存。刷新页面后所有数据都会丢失。请及时导出需要保存的内容。

#### Q4: 可以商用吗？
**A:** 项目采用MIT许可证，可以自由使用和修改。但请注意第三方库（PDF.js和Tesseract.js）的许可证。

## 性能优化建议

### PDF OCR性能优化

1. **选择合适的PDF**
   - 文件大小控制在10MB以内
   - 页数不超过20页
   - 分辨率适中（300dpi即可）

2. **浏览器优化**
   - 使用Chrome浏览器（性能最佳）
   - 关闭不必要的标签页
   - 清除浏览器缓存（如果遇到问题）

3. **设备优化**
   - 使用性能较好的设备
   - 确保有足够的可用内存
   - 关闭其他占用资源的程序

### 生日贺卡性能优化

1. **文件大小**
   - 照片控制在5MB以内
   - 音乐控制在10MB以内
   - 大文件会影响加载速度

2. **浏览器建议**
   - 使用硬件加速
   - 全屏观看效果最佳
   - 关闭浏览器插件（可能影响动画）
