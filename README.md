# PDF OCR 文字提取工具

一个基于浏览器的纯前端 PDF 文字提取工具，支持 OCR（光学字符识别）功能。

## 功能特点

- **多语言支持**：支持简体中文、繁体中文、英语及其混排识别。
- **本地处理**：所有处理均在您的浏览器中完成，文件不会上传到任何服务器，确保数据隐私安全。
- **批量处理**：支持同时上传多个 PDF 文件进行批量文字提取。
- **实时预览**：处理过程中可实时预览提取出的文字内容。
- **导出功能**：一键导出提取结果为 TXT 文本文件。
- **进度显示**：清晰显示当前处理的文件、页码及总体进度。

## 技术实现

- **PDF.js**：用于在浏览器中解析和渲染 PDF 页面。
- **Tesseract.js**：强大的开源 OCR 引擎的 JavaScript 移植版。
- **Canvas API**：用于处理 PDF 页面的图像转换。

## 使用说明

1. 点击“选择 PDF 文件”按钮，选取一个或多个 PDF 文件。
2. 在下拉菜单中选择 PDF 包含的主要语言。
3. 点击“开始提取文字”按钮。
4. 等待处理完成后，可以在预览区域查看结果。
5. 点击“下载 TXT 文件”保存结果。

## 浏览器兼容性

建议使用最新版本的 Chrome、Edge 或 Firefox 浏览器以获得最佳性能。
